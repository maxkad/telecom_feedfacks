Project: Analysing user Feedbacks

ChatGPT based system to query users feedback data.

How to use?

1) Go to .env and set up the below keys for OpenAI and for Pinecone (Vector DB):

OPENAI_GPT_4O_API_KEY
PINECONE_API_KEY

2) clone my repository
git clone https://github.com/maxkad/telecom_feedfacks

3) Install Python dependencies:
requirements.txt

pip install -r requirements.txt

4) Upload and prepare data: 
Create SQLite database and tables. 
Create Pinecone index:

Run prepare_and_upload_data.py
------------------------------
python scripts/prepare_and_upload_data.py
(It might take a few minutes)

- The tables exist in the repo.

5) Use streamlit to submit your queries.

Start streamlit:
----------------
streamlit run streamlit_app.py

Go to: http://localhost:8501

To be able to submit queries from the browser.
